# [荒岛解密](./index.html) 攻略
## 文字教程
1. 随机选择一个角点击直到与其对角相等

2. 选择第一步的邻角点击直到其他三角相等

3. 选择第二步的对角点击直到四个角相等

4. 依次点击四个角直到与中间相等
